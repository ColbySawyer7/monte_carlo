# Profiles Directory

This directory contains flamegraph profiling outputs generated by the profiler.

## Usage

### Option 1: Using Docker Compose (Recommended)

Run profiling directly with docker-compose:

```bash
# Generate DES flamegraph
docker-compose --profile profiler run --rm profiler-des baseline.json

# Generate Monte Carlo flamegraph
docker-compose --profile profiler run --rm profiler-monte baseline.json 500000
```

### Option 2: Using the Shell Script

Run the profiler script from `sim-native/`:

```bash
# Generate DES flamegraph
./profiler.sh des baseline.json

# Generate Monte Carlo flamegraph
./profiler.sh monte baseline.json 50
```

## Output Files

Flamegraphs are saved with datetime-based names:
- DES: `flamegraph-des-YYYYMMDD-HHMMSS.svg`
- Monte Carlo: `flamegraph-monte-YYYYMMDD-HHMMSS.svg`

## Viewing Flamegraphs

Open the SVG files in any modern web browser to view the interactive flamegraph visualization.
